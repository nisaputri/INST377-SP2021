Atom vs VSCode

  • InternetingIsHard.com, our textbook, uses Atom rather than VSCode. From this we can see that it was written a few years ago, by someone heavily involved in the internet.
    • Atom is from 2014, and is made by Github.
    • VSCode is from 2015, and is made by Microsoft.
  • As of 2018, Microsoft owns Github.
  • Microsoft now owns both VSCode and Atom.
  • All examples that refer to Atom will work almost identically in VSCode.
  • Stackoverflow.com publishes an annual state-of-the-web survey.

Readings Week 1

Following along with the readings, create a new, correctly-structured webpage in your copy of the course repository. Commit your code changes as you make them, so that you can roll them back if you make an error.

  • Your page must satisfy the following requirements:
    1. Contains only valid HTML
    2. Contains a page title with your name in it
    3. Properly uses h2 tags per section header
    4. Properly uses paragraph tags
    5. Contains an h1 header element with your name in it
    6. Contains an unordered list with three list elements in it
    7. Contains an ordered list with five elements describing your top 5 favourite albums
    8. Contains one picture of each: .jpg, .png, .svg, and a .gif - these are supplied in the tutorial
    9. Contains an emphasized element
    10. Contains a strong element
    11. Contains a strongly emphasized element
    12. Contains a hard rule
    13. One soft line break

  • Push your changes to the main branch of your lab repository
