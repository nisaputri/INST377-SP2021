Required Software List

Please note, this is not a "recommended" list - it is required to complete class exercises successfully with the full support of the instructors.

Node.js

  • You MUST have a minimum of Node v.14 installed for this course.
  • https://nodejs.org/en/download/

There are other ways to manage this install - notably, you can use a package manager like apt if you're on Linux or brew on Mac, but they tend to be tricky the first time out.

Git Client

  • Github Desktop
  • An account at Github.com
    • With a professional name, such as your first name, last name, and a number.
    • The instructional team would prefer not to receive Github accounts with names like xXx_my_bloody_romance_xXx this term.

VSCode Text Editor

  • VSCode https://code.visualstudio.com/
  • Required Extensions
    • Rainbow Brackets
      • JS brackets are tough for most people. This will make them easier to see.
    • Prettier, The Code Formatter
      • This will let you format your HTML/CSS/JS by right-clicking in VSCode and selecting "Format Document"
  • ESLint installed globally in your computer
    • This will help with JS formatting problems.
  • VSCode will, ideally, help you learn to program. Here's how, for HTML: 
    • https://code.visualstudio.com/docs/languages/html
    • Please review the use of Emmet for HTML assistance: https://code.visualstudio.com/docs/editor/emmet

FAQ

1. Can I Use A Different Text Editor

Sure, Atom and Sublime are both fine, but we can't help you with things in them. Vi, nano and emacs are out of bounds.

2. Can I use the Go Live server extension

Sure, but you will have a much harder time with labs. You learn to write basic servers in this course.

3. Can I Use Django

No. We learn a different type of web development in this course.
